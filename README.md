# Directory Size Calculator
A minimal cross-platform Directory size utility.


## Features
- Get total directory size by default
- Get all children file size plus total directory size if -v flag is used


## Build
### POSIX / Linux / macOS
1. `cd Directory-Size/`
2. `make`


### Windows
1. install MinGW64 (https://www.mingw-w64.org/)
2. using MinGw64 terminal:
	1. `cd Directory-Size/`
	2. `make`
